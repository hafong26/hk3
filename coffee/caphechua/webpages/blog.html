Bài tập lớn tháng 12

Tháng 11, 2021 bởi Nguyễn Văn Hào

Do tình hình căng thẳng bởi dịch Covid 2021 gia tăng chóng mặt nên nhà trường quyết định cho sinh viên học online và làm bài tập lớn để chấm điểm thi.

Nói chung thì cũng không đòi hỏi cao, vừa đủ sức với Sinh viên :))

Thiết kế website với Bootstrap và Jquery.

Giới thiệu sương sương về Bootstrap và Jquery

  • Bootstrap là một framework bao gồm các HTML, CSS và JavaScript template dùng để phát triển website chuẩn responsive.
  • Jquery là thư viện được viết từ JavaScript, jQuery giúp xây dựng các chức năng bằng Javascript dễ dàng, nhanh và giàu tính năng hơn

Yêu cầu nho nhỏ

  • Sử dụng grid của bootstrap để tổ chức responsive cho trang.
  • Sử dụng typography để định dạng text color, text align, text transform, heading
  • Thực hiện các thành phần giao diện:
    1. Header: logo, banner và các thông tin quan trọng như: tên thành viên, link giỏ hàng, link đăng ký/đăng nhập
    2. Menu: đặt các link chính của website
    3. Aside: các thông tin phụ, đặt bên phải hoặc bên trái trang (form search, form login, tin xem nhiều, hình sự kiện, sản phẩm bán chạy)
    4. Content: nội dung trên trang layout tương ứng với chủ đề
    5. Footer: hiện mô tả website, thông tin liên hệ, email, điện thoại, thống kê, link mạng xã hội…
    6. Các thành phần khác: button, card, accordion, heading, icon, carousel, form, list group, modal, table, badge, alert, hình ảnh…
  • Thực hiện các trang: trang chủ (index.htm), trang login, trang giới thiệu, trang sản phẩm,… nói chung càng nhiều càng tốt.

Inline HTML elements

HTML defines a long list of available inline tags, a complete list of which can be found on the Mozilla Developer Network.

  • To bold text, use <strong>.
  • To italicize text, use <em>.
  • Abbreviations, like HTML should use <abbr>, with an optional title attribute for the full phrase.
  • Citations, like — Mark Otto, should use <cite>.
  • Deleted text should use <del> and inserted text should use <ins>.
  • Superscript text uses <sup> and subscript text uses <sub>.

Most of these elements are styled by browsers with few modifications on our part.
